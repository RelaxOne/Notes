### MarkDown 笔记 Typora 使用教程

* __基本操作__

  * 标题用 ###（个数代表标题的级数）

  * 引用内容使用 >（个数表示引用的层数）

  * 代码用\`\` ：`string str = "hello..."`

  * 多行代码用\`\`\` 

    ```c
    int a  = 10;
    int b = 10;
    ```

  - 无序列表用 * + - 三种符号（多级加 tab）
  - 有序列表直接用 数字加小数点（多级加 tab）
  - 斜体用两个 \*\* 包括：*斜体*
  - 粗体用两个\__\__ 包围：__加粗__
  - 下划线用<u> </u> 包围：<u> 下划线 </u>
  - 删除线用\~~\~~包围：~~删除线~~
  - 分割线用三个连续的\ *,\-,\_:

* __链表__

  * 图片\!\[pic_name\]\(url\)的方式：
  * 超链接
    - 用两对[]，第一对中存显示连接名，第二对中存链接地址：[百度](http://www.baidu.com)
    - 直接用< > 将地址包括：<http://www.baidu.com>
